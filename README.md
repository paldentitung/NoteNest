# NoteNest

NoteNest is a simple note management application built with **React** (frontend) and **Node.js + Express** (backend). It allows users to create, edit, delete, and view notes with optional file attachments (PDF or image).

## Features

- Add new notes with title, subject, semester, description, and file.
- Edit existing notes with optional file update.
- Delete notes.
- View all notes.
- Responsive UI with React and Tailwind CSS.
- Local JSON file storage for backend (`data/data.json`).

## Tech Stack

- Frontend: React, Tailwind CSS, React Hot Toast
- Backend: Node.js, Express, Multer (for file uploads)
- Storage: JSON file
